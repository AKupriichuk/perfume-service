A web application for managing a perfumery database. The project allows users to track fragrances, their creators (perfumers), and manufacturers.

## Core Features

* **Fragrance Catalog**: View a list of scents and detailed information about them.
* **Perfumers**: A database of professionals who create unique fragrances.
* **Manufacturers**: Information about global perfume brands and producers.
* **Admin Panel**: Ability to add, edit, and delete data records.

## Tech Stack

* **Backend**: Python 3.8+, Django 4.2.
* **Frontend**: Soft UI Design System (Bootstrap 5).
* **Database**: SQLite3.
* **Tools**: PyCharm, Git.

## Local Setup

1. **Clone the repository**:
```bash
git clone <repository_url>
cd perfume-service

```


2. **Set up the virtual environment**:
```bash
python -m venv venv
source venv/bin/activate  # For Windows: venv\Scripts\activate
pip install -r requirements.txt

```


3. **Run migrations**:
```bash
python manage.py migrate

```


4. **Start the server**:
```bash
python manage.py runserver

```


The application will be available at: `http://127.0.0.1:8000/`.
